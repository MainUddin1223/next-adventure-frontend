# Next Adventure

### Live link

```
https://next-adventure-three.vercel.app
```

#### It's a tour and travel management platform web application. Using this platform tour and plan orginazer agency can create their profile and list their tour & plan services. User can see all the tour plan and can book their accrding to their test. User or Organizer can confirm,Reject or Cencel their booking before the plan deadline.

## Technology

- Nextjs
- Expressjs
- Nodejs
- Postgresql
- Prisma
- Redux

#### Beside above techonology React hook form, Joi Validation , Authorization and authentication , Ant design has been used to develop the project
